FAQ

What is FYIPedia?

An open-source ecosystem of 53 reference sites organized into 11 thematic families. Each site has a public REST API, npm embed package, and per-site widget docs. MIT licensed.

How many widgets are available?

87 npm packages (one per site) with 10+ widget types each = 870+ unique widget variations. Plus 10 family-level hubs (this page is the umbrella).

Are FYIPedia widgets free?

100% free. No paywall, no API key, no rate limit. MIT licensed. Widget footer shows "Powered by [site]" — the only requirement. Hosted on free CDN (jsDelivr).

How is the data sourced?

Each site sources from authoritative origins: IUPAC + NIST (chemistry), IUCN + eBird (wildlife), POWO + Kew (plants), USDA + DrugBank + FDA (health), Hipparcos + Gaia (astronomy), IMA (minerals), GIA (gems), ICANN + RFCs (network), IATA + ICAO (transport), MaxMind GeoLite2 (IP). Quarterly refresh.

Can I contribute or fork?

Yes — all source repos at github.com/fyipedia. Per-site embed packages at fyipedia/{site}-embed. Family hubs at fyipedia/{family}-widget-hub. PRs welcome.

What makes FYI Families different?

Three differentiators: (1) Cross-family linking — a font widget can reference unicode coverage, a chemistry widget can link to a mineral. (2) Consistent API + widget pattern across all 53 sites. (3) Open data + open source — embed without licensing fees, vendor lock-in, or API limits.
